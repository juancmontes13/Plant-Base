# Ecommerce React 39530

\_Proyecto final CoderHouse React

## Requerimientos

\_es necesario instalar las dependencias figuradas en el package.json

```
npm i
```

## Ejecucion

\_En la consola escriba el siguiente comando:

```
npm start
```

\_Luego vaya al navegador a la ruta:

```
localhost: 3000
```
